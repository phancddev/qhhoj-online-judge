Demo Images for DMOJ User Activity Tracker
=============================================

This directory should contain the following demo images:

1. demo1.png - Screenshot of the Active Users Dashboard
2. demo2.png - Screenshot of the User Activity Detail page

These images are referenced in the README.md and QUICK_START.md files.

To add your own screenshots:
1. Take a screenshot of the Active Users Dashboard page (/admin/user-activity/active-users/)
2. Take a screenshot of a User Activity Detail page (/admin/user-activity/user/{username}/)
3. Save them as demo1.png and demo2.png respectively in this directory
4. Make sure the images are clear and showcase the features of the application
